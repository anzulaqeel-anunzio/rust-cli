# Rust CLI Tool Template

A robust starter template for building high-performance Command Line Interfaces (CLI) using Rust.

## Features

-   **Clap**: Industry standard for command-line argument parsing.
-   **Anyhow**: Idiomatic error handling.
-   **Colored**: Terminal color support for better UX.
-   **Structure**: Best practices for binary crates.

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/anunzio-international/rust-cli-template.git
    cd rust-cli-template
    ```

2.  **Build**:
    ```bash
    cargo build --release
    ```

3.  **Run**:
    ```bash
    cargo run -- --help
    ```

4.  **Run with Arguments**:
    ```bash
    cargo run -- greet --name "Anunzio"
    ```
